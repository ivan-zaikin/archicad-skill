# archicad-skill

Плагин для Claude, который умеет работать с открытым проектом **Archicad 25**:
считать площади и объёмы, составлять экспликации помещений, ведомости отделки,
спецификации окон/дверей, квартирографию, ТЭП и другие документы по ГОСТ 21.501
и СП 54.13330.

---

## Установка (пошагово)

### Шаг 1. Открыть терминал

Нажмите **Win + R**, введите `powershell` и нажмите **Enter**.
Откроется окно с синей или чёрной рамкой — это и есть терминал.

### Шаг 2. Установить Python

1. Перейдите на [python.org/downloads](https://www.python.org/downloads/) и скачайте последнюю версию
2. Запустите установщик
3. **Обязательно** поставьте галочку **«Add Python to PATH»** на первом экране
4. Нажмите «Install Now»

Проверка в терминале:
```
python --version
```
Должно появиться что-то вроде `Python 3.12.4`.

### Шаг 3. Установить Node.js

Нужен для установки Claude Code CLI.

1. Перейдите на [nodejs.org](https://nodejs.org/) и скачайте версию **LTS**
2. Запустите установщик, везде нажимайте «Next»

Проверка:
```
node --version
```

### Шаг 4. Установить Claude Code CLI

Сначала разрешите выполнение скриптов в PowerShell (один раз):
```
Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned
```
На вопрос ответьте `Y` и нажмите **Enter**.

Затем установите Claude Code:
```
npm install -g @anthropic-ai/claude-code
```

Затем запустите и войдите в аккаунт:
```
claude
```

Следуйте инструкциям на экране — откроется браузер для авторизации через Anthropic.

### Шаг 5. Установить Claude Desktop

1. Перейдите на [claude.ai/download](https://claude.ai/download) и скачайте приложение для Windows
2. Установите и войдите в аккаунт

Claude Desktop — основной интерфейс для общения с Claude.

### Шаг 6. Установить плагин

Нажмите **Win + R**, введите `powershell`, нажмите **Enter**. Запустите Claude Code:
```
claude
```

Затем выполните две команды прямо в чате:
```
/plugin marketplace add ivan-zaikin/archicad-skill
/plugin install archicad-skill@archicad-skill
```

## Обновление плагина

1. Нажмите **Win + R**, введите `powershell`, нажмите **Enter**
2. Запустите Claude Code:
```
claude
```
3. Выполните в чате:
```
/plugin marketplace update archicad-skill
```

## Использование

1. Запустите Archicad 25 и откройте проект
2. Откройте PowerShell (**Win + R** → `powershell` → **Enter**) и запустите:
   ```
   claude
   ```
3. Напишите что-нибудь вроде: **«сделай экспликацию помещений»** или
   **«посчитай объём бетона»** — Claude сам разберётся

> **Важно:** плагин работает только в терминале через `claude` (Claude Code).
> В обычном Claude Desktop (приложение) скилл недоступен.

---

## Что умеет

- Экспликация помещений, ведомость отделки, спецификация окон/дверей
- Квартирография, ТЭП, экспликация полов
- Объёмы бетона и кладки с разбивкой по конструкциям
- Проверка данных: валидация зон, кросс-проверка площадей
- Классификации, пользовательские свойства, навигатор

## Ограничения

- Только чтение данных + запись свойств/классификаций; геометрию менять нельзя
- Полигон зоны API не отдаёт (только bounding box) — для точных площадей
  нестандартных форм нужен IFC-экспорт
- Поддерживается AC25; часть команд из новых версий Archicad недоступна
